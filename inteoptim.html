Optimization

process of finding maximum and minimum values given constraints using calculus

Steps

  1. list the given value determine max or min
  2. Write a equation and find the derivative
  3. Test second derivative (to prove max or min)
  4. substitute back into equation

Explain

  • Determine what its asking (max)
  • Set up equation
  • Find derivative and set to 0
  • Test the point is max or min
  • plug back to the equation

Key

There are two way to test the max or min
  • Find second derivative and plug the x-value
  • Used first derivative plug a number bigger than and less than the x-value and use the first derivative test
