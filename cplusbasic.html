1.3 Variable and Constant

  • Variable = given a name to a set of data
  • Rule : data type + data name = variable detail
  • EX.
    #include <iostream>
    using namespace std;
    int main()
    { //Rule : data type data name = variable detail
    int a = 10
    cout << "a= " << a << endl;
    system("pause");
    return 0; }
  • Constant = data name that will not be change in the project
  • Rule : #define + macro constant
    const + data type + data name = data detail
  • Ex
    #include <iostream>
    using namespace std;
    #define day 7 int main()

    cout << "in a week there is " << day <<"day" <<endl;
    system("pause");
    return 0; }
