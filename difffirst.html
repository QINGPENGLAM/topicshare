First derivative test and Second-derivative test

Need to know first (Sign is very important)

  • f(X)is increasing when f' positve
  • f(x) is decreasing when f' negative

First Derivative Test (f(x) is continuous then at critical point,c)

  • If f' change from pos to neg then f has local max at c
  • If f' change from neg to post then f has local min at c
  • If f' does not change at c then f has no max or min

Concavity and inflection points

  • Concave down = f'' negative
  • Concave Up = f'' postive
  • Inflection point = (at critical point C) f'' change sign

Second Derivative Test

  • If f'(x)=0 and f'' negative then f has local max at x=c
  • If f'(x)=0 and f'' postive then f has local min at x=c

Quick take away

  • f'' pos = local min neg = local max
  • concave up = pos down= neg
  • f' pos-neg = max neg-pos= min
  • f' pos= f increase f' neg= f decrease
