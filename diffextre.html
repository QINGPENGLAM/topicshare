Need to know first

  • Absolute maximum =(f(c) >f(x) for all x in function)
  • Absolute Minimum =(f(c) smaller than all x in function)
  • Relative (local) maximum = absolute maximum near c (limit)
  • Relative (local) Minimum= absolute minimum near c (limit)

Extreme Value Theorem

  • If f is continuous on closed interval then f has both max and min value on the interval

Key

  • If open interval it will be close to the max or min but never reach it
  • One close one open resulted only one extreme value

Critical point (x-value)

  • F prime=0 or f prime DNE

How to find extreme value

  1. Find critical point
  2. Find Y value of critical point and end point
  3. Biggest Y is max Smallest Y is min
