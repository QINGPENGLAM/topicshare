Continuity

By definition
If a function has all the point on the line
(it will be used in differentiable and derivative)

Checklist (Important)

  • Lim (x to C) F(x) exists( limit exist)
  • F(c) exist (y-value exist)
  • If Above is equal (limit equal to value)
  • The point is continuity

Explain

  • When given expression you plug the inequality
  • Check if both limit is equal
  • If true then continuous if False its not continuous
  • sometime theres a given f(x)value
  • If ask f(3) look for equal sign in the inequality
